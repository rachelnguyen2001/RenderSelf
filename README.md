# RenderSelf
A Physically-Based Makeup & Skin Tone Simulation

## 📖 Overview

RenderSelf is a computer graphics application designed to provide a high-fidelity "digital canvas" for makeup and skin tone experimentation. By leveraging ray tracing, the program goes beyond simple color overlays to simulate how light interacts with different pigments and surface textures on a human face.

The project allows users to manipulate skin undertones, lip pigments, and eye colors through precise adjustment levels, providing a realistic preview of how various cosmetic choices harmonize.

## 🚀 Key Features

Ray-Traced Rendering: Utilizes recursive ray casting to calculate realistic shadows, reflections, and light diffusion across facial contours.

Dynamic Skin Tone Mapping: Adjustment layers that simulate melanin levels and undertone shifts (cool, neutral, warm).

Material Properties: Distinct shaders for different surfaces—matte finishes for skin, high-gloss specular reflections for lip color, and refractive caustic effects for the iris.

Interactive UI: Real-time sliders for fine-tuning "adjustment levels" with instant visual feedback.
